# 🐾 Plataforma Digital de Adopción

## 📌 Descripción del Proyecto

La Plataforma Digital de Adopción es un Producto Mínimo Viable (MVP) desarrollado para facilitar el proceso de adopción de mascotas mediante una aplicación web.

El sistema permite a los usuarios registrarse, iniciar sesión, visualizar mascotas disponibles y gestionar solicitudes de adopción.

---

## 🎯 Objetivos

### Objetivo General

Desarrollar un MVP funcional que entregue valor de negocio, implementando pruebas unitarias, integración continua (CI/CD) e inspección de calidad del código.

### Objetivos Específicos

* Implementar pruebas unitarias
* Configurar CI/CD con GitHub Actions
* Simular análisis con SonarQube
* Desarrollar funcionalidades del sistema

## ⚙️ Instalación y Ejecución

### 1. Clonar repositorio

git clone https://github.com/abbeygonzlz/plataforma-digital-adopcion.git
cd plataforma-digital-adopcion

### 2. Instalar dependencias

npm install

### 3. Ejecutar proyecto

npm run dev

(o en su defecto)
npm start

### 4. Acceder

http://localhost:5173 (Puede variar)

## 🔄 CI/CD

Se implementó un pipeline con GitHub Actions que realiza:

* Build del proyecto
* Ejecución de pruebas
* Análisis de seguridad (npm audit)
* Simulación de análisis con SonarQube
* Deploy simulado

El pipeline se ejecuta automáticamente en cada push.
